LsqFit.jl

Basic least-squares fitting in pure Julia under an MIT license.

The basic functionality was originaly in Optim.jl, before being separated into this library. At this time, LsqFit only utilizes the Levenberg-Marquardt algorithm for non-linear fitting.

LsqFit.jl is part of the JuliaNLSolvers family.

SourcePackage EvaluatorBuild Status

Install

To install the package, run

Pkg.add("LsqFit")

If you want the latest features, also run

Pkg.checkout("LsqFit")

To use the package in your code

julia> using LsqFit
